Instalación de JDK en los sistemas operativos Windows 10 y Debian 8

Java es un lenguaje de programación y una plataforma informática comercializada por primera vez en 1995 por Sun Microsystems. Hay muchas aplicaciones que usan Java el cual es rápido, seguro y fiable. Desde portátiles hasta centros de datos, desde consolas para juegos hasta súper computadoras, desde teléfonos móviles hasta Internet, Java está en todas partes.

Ir Pagina para descargar JDK

Visitar pagina oficial de JDK 8

Se busca el sistema operativo donde se planea instalar en este caso descargaremos de Windows y Debian

Se necesita iniciar sección para poder descargar los ejecutables.

Ejecutar instalador de jdk en Windows

Ejecutamos el programa descargado y damos a siguiente, ya que las fuciones, en default no son necesarias de cambiar.

Ejecutar instalador de jdk en Debian

Para esto abrimos la terminal iniciamos secion en super usuario usando su y escribimos:
echo "deb http://ppa.launchpad.net/webupd8team/java/ubuntu xenial main" | tee /etc/apt/sources.list.d/webupd8team-java.list
Despues de ejecutar este comando ejecutamos:
echo "deb-src http://ppa.launchpad.net/webupd8team/java/ubuntu xenial main" | tee -a /etc/apt/sources.list.d/webupd8team-java.list
El siguiente comando es:
apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys EEA14886
Ahora actualizamos con:
apt-get update
Lo siguiente es iniciar la instalacion con el comando:
apt-get install oracle-java8-installer
Por ultimo se ejecuta:
apt-get install oracle-java8-set-default
