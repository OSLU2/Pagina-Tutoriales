Instalación de Windows 10

El sistema operativo windows 10 es la última verión estable del sistema de Microsoft.
En esta página aprenderas a instalar el sistema operativo desde una USB desde cero a una computadora.

Ir descargar herramienta Windows 10

Crear medio de instalación

Se inicia la herramienta que windows nos proporciona para crear el medio de instalación del sistema operativo.
Se indica si se desea instalar por medio de USB o disco, en este caso sera USB.

Se espera a que se termine de instalar el medio de instalación de windows 10.

Ejecutar USB en computadora

Se busca en la BIOS poner de prioridad en boot la memoria USB.
Esperamos un poco para que empieze arrancar y se elige el idioma, el formato y el teclado.

Se espera a que se termine de instalar el medio de instalación de windows 10.

Elegir Versión de Windows 10

Si se tiene una clave para activar el Windows 10 se puede agregar, de lo contrario se brinca.
Selecionamos la versión a usar de Windows 10 en mi caso usare el pro.

y se aceptan terminos y condiciones.

Elegir particiones

Se elige la opción personalizada donde elegimos la partición
podemos formatear para mejor rendimiento del sistema operativo, al menos es mi recomendación

Si se decicio formatear es porque se hizo respaldo de la información importante.
Se selecciona una partición y se espera a que termine la instalación.
