windows Server 2019 ofrece funciones destinadas a mejorar el rendimiento de la red y opcion de multiples colas en la maquina vistual dinamica.

Instalar Windows Server 2019

Para instalar Windows Server descargaremos la iso en la pagina oficial.
Instalamos la iso en una memoria y la ejecutamos en una computadora. lo siguiente es elegir la version en este caso usaremos la version standar evaluation con experiencia de escritorio, despues elegimos el espacio para instalar. y esperamos a que termine la instalación. Despues ponemos una contraseña para entrar.
Ya que estamos adentro se abrira automaticamente el administrador del servidor. nos vamos a la opcion que dice herramientas en la parte superior derecha y seleccionamos administración de equipos.
Buscamos Usuarios y grupos locales y damos clic derecho en usuarios para crear uno nuevo. En este caso creamos 10 usuarios. Nos vamos ahora en la opcion de grupos y damos clic derecho para crear un grupo nuevo, para agregar un usuario elegimos la opcion de agregar y escribimos en el area de texto el nombre de usuario para comprobar de clic en comprobar nombres.
