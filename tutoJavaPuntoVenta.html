Ejecución de punto de venta de venta Java

Java tiene un codigo que puede ser ejecutado en linux o Windows, solo se tiene que tener la version de java en los sistemas operativos.
El siguiente link te lleva al deposito de github donde se encuentan el codigo y el archivo .jar.

Ir Pagina de libreoffice

Ejecutar el archivo .jar

Para abrir el ejecutable en windows solo se necesita dar doble clic al archivo. Para ejecutar el .jar en Debian entras a la terminal y entras a la carpeta en donde esta el archivo y escribes el siguiente comando: java -jar PuntoVenta.jar
