Instalar Ruby on rails en Windows 10

Para instalar Ruby on Rails descargaremos un instalador para instalar Ruby, Rails, Sqlite, bundler, devkit,tinyTDS, SQL server support y git en caso que no lo tenga.
Se inicia el ejecutable y si desean pueden instalar git tambien en mi caso lo omito puesto que ya esta instalado.
Esperamos a que termine de instalar y ejecutamos la consola o cmd el siguiente comando:
'gem --version' para verificar.
Lo siguiente es escribir 'gem install bundler' para actualizarlo.

Entramos a la consola para escribir 'rails new proyecto' en la carpeta que queramos.
Abrimos el archivo 'Gemfile' para cambiar el gem 'sqlite3' por gem 'sqlite3', '~> 1.3.6' y guardamos.
Ahora entramos a la direccion del proyecto y ejecutamos bundle install. para el pryecto crearemos una tabla llamada alumnos ejecutando el siguiente comando en la consola
'rails g scaffold alumnos Nombre:string direccion:string telefono:string fechadeMatriculacion:date'
luego migramos el proyecto con 'rails db:migrate' y ejecutamos el servidor con 'rails s', donde para entrar ejecutamos el navegador y escribimos 'localhost:3000/alumnos'
